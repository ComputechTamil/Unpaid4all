1. Install Revit Structure Suite 2009.

2. Start Autodesk Revit iStructure 2009 and Select Activate the Product 
   when ask for activation,  select Enter an Activation Code
   Use as serial : 653-12354321 or 666-98989898 or 666-69696969
   Write down the request code.
   Or from Help menu, select Product Ind license Information and choose Standlone-Locked

3. Fire up the xforce keygen and generate an activation code
   based on your request code. Continue the registration process
   Copy and paste your generated activation code ..
   Restart Autocad

4. Enjoy!



We have included 2 "special" keygens for Autocad 2009 part of this Revit Suite
if you want to re-activate or just install Autocad
But if you registered Revit Structure you dont need to activate Autocad 2009
just install it and you are done
