1. Install Revit MEP 2009.

2. Start Autodesk Revit MEP 2009 and Select Activate the Product 
   when ask for activation,  select Enter an Activation Code
   Use as serial : 653-12354321 or 666-98989898 or 666-69696969
   Write down the request code.
   Or from Help menu, select Product and license Information and choose Standlone-Locked

3. Fire up the xforce keygen and generate an activation code
   based on your request code. Continue the registration process
   Copy and paste your generated activation code ..
   Restart Revit

4. Enjoy!

Autocad MEP is 64bit only (dont ask us why, original dvd has 64bit only)
so install first Revit MEP, register it and then install Autocad MEP if you want
